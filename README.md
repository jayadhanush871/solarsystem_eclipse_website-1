# solarsystem_eclipse_website
The Solar System Eclipse Website is a simple educational webpage that explains planets and eclipses using basic HTML, images, and audio. It helps students learn about the solar system in an easy and interactive way. The site is designed for beginners and shows how to create a basic science website without advanced coding.
# Project Summary
The Solar System & Eclipses Interactive Website is a simple educational
project created to help students understand basic astronomy concepts
in an easy and interactive way. The main idea behind this project is to
make learning about the Solar System and eclipses more interesting and
less confusing for beginners.
The website starts with Login and Registration pages, which allow users
to access the system securely. After logging in, users are taken to the
Home page, where they get a brief introduction to the Solar System,
including the Sun, planets, moons, asteroids, and comets. This page
helps users understand what the website is about before moving
forward.
The Learning page allows users to explore different topics related to the
Solar System and eclipses. Users can select topics and read simple
explanations that are easy to understand. This section is designed
mainly for beginners, so the content is kept clear and straightforward.
Visual placeholders are also included to make the page look engaging
and organized.
To check how well the user has understood the concepts, the website
includes an Assessment page with multiple-choice questions. After
answering the questions and submitting the assessment, the user is
shown an Assessment Result page, which displays the total number of
questions, correct answers, and whether the user has passed or failed.
# About Landing Page
The Landing page is the first page of the website and provides access to
the Login and Registration options. It allows new users to create an
account and existing users to log in easily. 
# About Home Page
The Home page acts as the starting point of the website and gives a
clear introduction to the Solar System and eclipses. This page helps
users understand the purpose of the website in a simple way. It also
guides users smoothly toward the learning and assessment sections.
Overall, the Home page makes the user feel comfortable and interested
in exploring the website further.
# About Learning Page
The Learning page is where users actually start learning about the Solar
System and eclipses. It presents the topics in a simple and easy way so
that beginners can understand without feeling confused. Users can read
and explore different concepts at their own pace. This page helps build
a strong basic understanding before moving to the assessment. It makes
learning feel comfortable and engaging.
# About Assessment Page
The Assessment page is used to check how well the user has
understood the learning content. It contains multiple-choice questions
related to the Solar System and eclipses. Users can select their answers
and submit the assessment easily. After submission, the results are
shown to help users know their performance. This page makes learning
more interactive and meaningful and can take quiz up-to 10 questions.
# About Feedback Page
The Feedback page allows users to share their opinions and suggestions
about the website. It provides a simple form where users can enter
their name and feedback message. This page helps understand the
user’s experience and areas for improvement. It also encourages user
interaction with the system. Overall, the Feedback page helps make the
website more user-friendly.
