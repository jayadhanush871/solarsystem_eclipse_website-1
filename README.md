# Solar System Eclipse Website 🪐

The **Solar System Eclipse Website** is an interactive educational portal built to simplify studying for students. This project demonstrates a full user journey—from initial registration to a personalized learning dashboard.

---

### 🚀 Key Project Features

* **User Access & Security**
    * **New User Registration:** A dedicated sign-up flow allowing new learners to create an account.
    * **User Login:** A secure gateway for existing users to enter the portal.
    * **Landing Page:** A professional entry point providing clear navigation to join or log in.

* **Educational Content**
    * **Interactive Home Page:** A welcoming introduction to the Sun, planets, moons, and other celestial bodies.
    * **Learning Hub:** Simplified explanations of complex topics like orbital mechanics and eclipses, curated for beginners.

* **Assessment & Interaction**
    * **10-Question Quiz:** A multiple-choice assessment designed to check knowledge retention.
    * **Instant Evaluation:** A results page that calculates scores and provides immediate "Pass/Fail" feedback.
    * **User Feedback System:** A dedicated form for users to share suggestions and improve the portal.

* **Personalized Profile Dashboard**
    * **User Information:** Dynamically displays the user’s account details from the registration process.
    * **Academic Tracking:** Automatically captures and stores scores from the Assessment Page.
    * **Data Persistence:** Utilizes **LocalStorage** to ensure that user progress and marks are saved across sessions.

---

### 🛠️ The Tech Behind the Project

* **React.js:** Used for building a component-based UI and managing application state.
* **JavaScript :** Powers the quiz logic, authentication flows, and dynamic data rendering.
* **HTML5 & CSS3:** Provides a responsive, "space-themed" structure and layout.

---

### 📂 How the App Flows
1. **Register/Login** ➔ 2. **Home** ➔ 3. **Learn** ➔ 4. **Quiz** ➔ 5. **Profile (View Marks)**
